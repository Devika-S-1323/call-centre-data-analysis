# Call Center Data Analysis Dashboard

## Overview

This project analyzes call center data using Excel and presents key insights through an interactive dashboard.

The dashboard helps track:

* Total calls received
* Customer satisfaction (CSAT) score
* Call duration
* Customer sentiment
* Call volume trends
* Call channels
* State and city-wise performance
* Common customer issues and reasons for calls

## Dataset

The dataset contains the following fields:

* Customer ID
* Customer Name
* Sentiment
* CSAT Score
* Call Timestamp
* Call Day
* Reason for Call
* City
* State
* Channel
* Response Time
* Call Duration (Minutes)
* Call Center

## Dashboard Features

### KPI Cards

* Total Calls
* Average CSAT Score
* Average Call Duration
* Response Time Metrics

### Visualizations

* Call Volume Trend
* Customer Sentiment Analysis
* Most Common Call Categories
* Average CSAT Score
* Average Call Duration
* Call Channel Distribution
* State-wise Analysis
* City-wise Analysis

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Dashboard Design

## Project Objective

The goal of this project is to analyze customer interactions and identify trends that can improve customer service performance and operational efficiency.
